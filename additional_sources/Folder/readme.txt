QML Chess Game

1. Original requirements are stored in src/doc/DAI_task.doc 
2. Documentation that explains architecture is available at doc/index.html
3. Application consists of qmlchesslib library and qmlchess executable. Library was created to simpily unit tests development.
4. Directory src/tests contains unit tests based on QtTest
5. Tested on 
- Windows 7 / MinGW32 / Qt 5.4 
- Linux OpenSuse 13.1 / g++-64 / Qt 5.4.2
